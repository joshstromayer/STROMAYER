Introduction to Objects

In JavaScript, objects are a fundamental way to store and manage structured data. Unlike primitive data types (numbers, strings, booleans, etc.), objects allow you to store multiple pieces of related data together.

What is an Object?

An object is a collection of key-value pairs. Each key is a property name, and its corresponding value can be any data type, including numbers, strings, arrays, or even other objects.

Example: Creating a simple JavaScript object

let person = {
    name: "Jordan",
    age: 25,
    isStudent: false
};
console.log(person);

In this example, the object `person` has three properties: `name`, `age`, and `isStudent`.

Why Use Objects?

Objects are useful when you need to group related data together. Instead of having separate variables, objects let you organize data in a structured way.

Example: Without objects vs with objects

// Without an object
let name = "Jordan";
let age = 25;
let isStudent = false;

// With an object
let person = {
    name: "Jordan",
    age: 25,
    isStudent: false
};

Objects Can Store Functions

Objects can also store functions as properties. These functions are called methods.

Example: Adding a method to an object

let car = {
    brand: "Toyota",
    model: "Corolla",
    year: 2022,
    start: function() {
        console.log("The car is starting...");
    }
};
car.start(); // Calls the method

Conclusion

Objects are one of the most important parts of JavaScript programming. They allow you to store complex data, define methods, and structure your programs in a more readable and maintainable way.

2. Accessing Object Properties

Given the object below, how would you access the person's age?

let person = {
    name: "Sophia",
    age: 30,
    job: "Engineer"
};
console.log(person.age); // Output: 30

3. Object with a Method

    brand: "Samsung",
