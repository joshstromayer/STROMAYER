JS Basic Array Methods

JavaScript provides built-in methods to add and remove elements from an array. In this lesson, we will explore the most commonly used methods for modifying arrays.

Adding Elements to an Array

Using `push()`

The push() method adds one or more elements to the end of an array.

Example: Adding elements using `push()`

let fruits = ["Apple", "Banana"];
fruits.push("Cherry");
console.log(fruits); // Output: ["Apple", "Banana", "Cherry"]

Using `unshift()`

The unshift() method adds one or more elements to the beginning of an array.

Example: Adding elements using `unshift()`

let colors = ["Blue", "Green"];
colors.unshift("Red");
console.log(colors); // Output: ["Red", "Blue", "Green"]

2. Removing Elements from an Array

Using `pop()`

The pop() method removes the last element from an array.

Example: Removing elements using `pop()`

let numbers = [1, 2, 3, 4];
numbers.pop();
console.log(numbers); // Output: [1, 2, 3]

Using `shift()`

The shift() method removes the first element from an array.

Example: Removing elements using `shift()`

let animals = ["Dog", "Cat", "Elephant"];
animals.shift();
console.log(animals); // Output: ["Cat", "Elephant"]

Conclusion

The push() and unshift() methods allow us to add elements, while pop() and shift() let us remove them. These methods are essential for working with arrays dynamically.
